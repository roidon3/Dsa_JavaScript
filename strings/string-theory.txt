1. How Strings are Stored in JavaScript
JavaScript stores strings as a sequence of UTF-16 code units.

Each character is represented internally by a numeric value.

For example:

let str = "Hi";
console.log(str.charCodeAt(0)); // 72  (H)
console.log(str.charCodeAt(1)); // 105 (i)


2. Strings are Immutable
Immutable means: once a string is created, it cannot be changed in place.

Example:

let str = "Hello";
str[0] = "J"; // ❌ This won't work
console.log(str); // "Hello" (unchanged)

Even though it looks like an array (str[0]), trying to change it does nothing.
When you "change" a string, JavaScript actually creates a new string in memory.

let str = "Hello";
str = "J" + str.slice(1); // ✅ Create a new string
console.log(str); // "Jello"

Here, "Jello" is a new string — "Hello" still exists in memory until garbage collected.

3. Why Strings are Immutable
Performance: Since strings are often reused, immutability helps with caching and memory optimization.
Security: Prevents accidental or malicious modifications of string literals.
Consistency: You can safely share the same string value across different parts of the code without worrying about it changing.


4. What Happens When You Modify a String
When you "modify" a string:
JavaScript allocates new memory for the new value.
The original string remains untouched.
Your variable now points to the new string.

let a = "Cat";
let b = a;     // b points to same string as a
a = "Dog";     // new string created for a
console.log(b); // "Cat" (unchanged)

let a = "Cat";
let b = a;     // b points to same string as a
a = "Dog";     // new string created for a
console.log(b); // "Cat" (unchanged)


5. Key Takeaways
Strings in JS are primitive types.
They are immutable — can’t be changed in place.
Any "modification" creates a new string.
Use string methods (slice, replace, concat, etc.) to create new strings when needed.

Problem Statement
Given a string, find the length of the longest substring without repeating characters.

Input: "abcabcbb"
Output: 3   // "abc"

Approach: Sliding Window
We use two pointers (start and end) to create a moving window over the string:

Move end forward adding characters to a set/map.

If a character repeats, move start forward until there are no repeats.

Track the max length along the way.

function lengthOfLongestSubstring(s) {
    let set = new Set();
    let left = 0;
    let maxLength = 0;

    for (let right = 0; right < s.length; right++) {
        while (set.has(s[right])) {
            set.delete(s[left]);
            left++;
        }
        set.add(s[right]);
        maxLength = Math.max(maxLength, right - left + 1);
    }

    return maxLength;
}

console.log(lengthOfLongestSubstring("abcabcbb")); // 3
console.log(lengthOfLongestSubstring("bbbbb"));    // 1
console.log(lengthOfLongestSubstring("pwwkew"));   // 3
